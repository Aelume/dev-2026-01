# TaskFlow 任务看板

## 项目简介

这是一个使用 `HTML + CSS` 实现的静态任务管理页面，包含仪表盘统计、任务列表和项目动态三个主要区域。

## 页面结构

- `index.html`：页面结构与内容
- `styles/main.css`：全局样式、响应式布局和主题配色
- `assets/`：预留静态资源目录

## 当前设计方向

页面主题采用清新淡雅的薄荷绿、雾青与米白组合，整体保持轻盈、克制和安静的视觉氛围。

## 实现要点

- 使用语义化标签组织页面结构，例如 `header`、`nav`、`main`、`section`、`article`、`footer`
- 使用 `grid` 布局承载仪表盘卡片和信息卡片
- 使用 `flex` 布局处理头部操作区和任务列表行内元素
- 使用 CSS 变量统一管理颜色、阴影和边框，便于后续维护
- 提供移动端搜索切换、任务状态标签和回到顶部入口

## 响应式说明

- `980px` 以下：仪表盘卡片调整为两列，头图改为单列
- `720px` 以下：顶部搜索切换为移动端交互，导航支持横向滚动
- `520px` 以下：仪表盘卡片改为单列，任务列表自动换行
